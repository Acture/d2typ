# docpack Refactor Design Set

This directory contains the Phase 0 design assets for the large-scale `docpack` refactor.

Reading order:

1. `architecture-rfc.md`
2. `cli-manifest-spec.md`
3. `backend-contract.md`
4. `migration-plan.md`

These documents are intended to be decision complete for implementation. If implementation diverges, update the design docs first instead of letting code become the new source of truth.
